# InfraGuard

InfraGuard is an AI-powered Industrial Safety and Structural Health Monitoring System built using YOLOv8, OpenCV, and Deep Learning.

It detects:

- PPE Compliance (Helmet, Vest)
- Worker Presence
- Danger Zone Intrusion
- Structural Crack Detection
